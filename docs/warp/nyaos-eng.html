What's NYAOS

NYAOS(Nihongo Yet Another Os/2 Shell) is the commandline shell for OS/2 like tcsh, bash, 4OS2 and YAOS. You can use NYAOS without no paying to anyone forever;-) `Nihongo' means the language Japanese, but NYAOS supports not only Japanese but also the other Double Bytes Character System(DBCS).
  • Command-line editor like tcsh/wordstar. It can be customized.
  • Understand '#!' like unix. Of course, BATCH and REXX can be executed.
  • Open the file or directory on Workplace Shell
  • Alias
  • History
  • `&' , `;' , `~' , foreach like csh
  • Colored ls which can prints comments and longname in EA.
  • Supports DBCS.
  • Filename/commandname completion like tcsh/bash.
  • Directory stack(pushd/popd/dirs)
NYAOS requires:
  • IBM OS/2 2.11 or later.(tested only on Warp 4)
  • emx runtime libraries 0.9d or later
