ワタシのカレーはパームパイロット〜 (99/11/6)

見
事に「 ひゅ〜〜〜ん, ・・ポトッ・・」っと落ちてしまった… 本当は「キューンキューン」と行きたいところなのだが(笑)。 で、どこへ落ちたかというと、WorkPad (ちなみに 8MB モデルぢゃないです、残念ながら)。 何がいいかって言うと文書ネタとか、フリーソフトのアイデアとか、 思いついた時にパっと書けることですね。

これで、更新頻度二倍？
それは無理

Graffiti文字による手書き入力って奴も、 入力する度にストレスがたまるのではないかと、 買う前にかなり心配したんだけど、全然Ｏｋであった。 ストレスどころか、何か入力するネタは無いかと、 妄想しまくる始末。難儀なやっちゃ。

で、使っていて実にセンスいいなぁと感じたのは、 ＰＣと同じように、スペースとか改行の Graffiti を書くと、 [変換]や[確定]とみなしてくれるところ。 おかげで、入力枠のところをほとんど見ずに書けますわぁ〜。 もう、この入力方法、ワタシの中ではＳＫＫ、かんなの 次くらいにランキングしているぞ。

実にいい買い物をしたもんだ

もう嬉しがって、電車の中とか、客先で使いまくり。 ガキですねぇ〜。 しかし、一番いいのは、 起動に全く時間がかからないところだったというのは秘密である (そもそも「起動」なんて言葉、似合わねーよな)。

真面目な話をすると、仕事にしろ、プライベートにしろ、 これでメモする習慣がついたのが実に嬉しいところだと思う。 正直、普通のメモ帳ってのは、机の無いところではかなり書き込みにくいし、 自分で書いた字の汚さに「ギャワ〜」ってなってしまう。 それで書かないもんだから、余計「ギャワ〜」ってなってたもんね。 これで、きっと「真・葉山」に成長できることであろう(どこが真面目な話だ)。

「真・葉山」になったら、 影山ヒロノブも唄ってくれるかな？
絶対無理。それ以前に「真・葉山」って何？

で、まぁ、そのまま単品で遊んでいても(やっぱ遊んでんぢゃねーか)、 50％も使っていないので、PCと継ぐことにする。 シリアルポートを空けるために、ジョーシンへ行って、PS/2 マウスを購入。 で、本体に差したのはいいのだが、 Windows の プラグ＆無礼に邪魔されて、結構、難航してしまった…。 一時的だが、CD-ROM さえ認識しなくなったのにはビックリ。 でも、BIOS で、PS/2 用の IRQ を「ISAで使うぞー」っていう設定にして、 何回かリブートしたら直った。よかったよかった。

で、Windows に付属ソフトをインストールした後で、次にしたことと言えば、 実は OS/2 用ソフトのインストールではなく、 Palm用の囲碁ソフトを入れたことであった。 なんせ、はまっていたからな。 で、さっそくやってみたのであるが…。 が、ヨワい…、もとい、弱いのである。 一局やっただけで、もう放り出してしまった。 まぁ、マシンパワーが違うから仕方ないかな。

自分より弱い奴を見付けて、 ちょっと嬉しかったりして…
レベルが低いぞ、レベルが

その後は当然 OS/2 用のソフトのセットアップをするのである。 Hobbes にある Pilot-link と、織華さんの白龍シリーズだ。 だが、これがちょっと今難航していたりしちゃったりしている。 よって、この話は次回へ《つづく》ということにしておこう(笑)
