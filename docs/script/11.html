ff 0.20 を公開しました！ でも、扱いはα２ということで、お願いします。 今回はページャを内蔵したのが目玉です。 つーても、karl をくっつけただけなんですけどね。

しかしながら、単体ページャの時は、ファイル閲覧が終了したら、 ヒープもスレッドも皆放りっぱなして exit してよかったけれども、 今回はそうはいかなかったので、それなりに手間はかかりました、 0.5人日ほど(多いような少ないような数値だな;-)

最初は新規にページャ部分を作ろうとしたんですけど、くじけちゃって。 DOS の miel が FD のページャ部分を独立させて作ったのど逆ですね。 ちなみに eff の方はページャとして使えるような仕様にしてあります。 

	eff document
  や
	type document | eff
なんて使い方もできちゃったりします。 もう、karl 要らないかな。

でも、実のところ、ページャは、eff のウインドウの一つで表示みたいな形にしたかったんですけどね。格好いいけど、それほど実用性ないかしら。

なんか、久々に開発関係のことを書いたような気が…。 まぁ、KashEDIT に触発されたということで。
