≡≡≡≡★

山
地師といえば圧縮ということで(彼は自他認める圧縮狂)、 bzip2 の OS/2 版が更新されましたね。 ここだったかな。

僕は自分のプログラムの配布サイズを結構気にするので、bzip2 はどんどん使いたい。 できることなら、バイナリパッケージは LHA , ソースパッケージは tar+bzip2 一本で配布できたらと思います。 ところが、ちょっと障壁があって、

  • FATを意識して拡張子を 8.3 に収めたい。が、bzip2 は tb2 とか tbz なんていう変則的な拡張を gzip と違って認めてくれない。 パイプを使えば関係ないが、ちょっと誰でも出来るとは限らない。
  • gzip ほど普及してないので、念の為、gzip と bzip2 の両方をウェブに 置きたくなる。そうすると、パッケージを作る手間が 1.5倍。
  • ベクターソフトウェアパックに bzip2 は無い。 そういえば、ベクターって、bzip2 は全く扱ってないのかしら。
気にしすぎですかねぇ。bzip2版だけ置いても、 自分で(僕のプログラムを)コンパイルをしたがるような人なら、 gzip版がなくても、問題ないかしら。
